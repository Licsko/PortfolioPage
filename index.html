Tapasztalatok

Nyelvek

  • Elsajátítottam a HTML 5 alapjait. Nem okoz gondot egy alap weboldal struktúráját felépíteni. Haladóbb dolgokat jelenleg is tanulok.

    HTML5

  • Elsajátítottam a CSS 3 alapjait. Nem okoz gondot egy oldal stílusát elkészíteni. Haladóbb dolgokat jelenleg is tanulok, mint például az értékek változóba mentése. Tervemben van a SASS CSS megtanulása is.

    CSS3

  • Elsajátítottam a JavaScript alapjait. Jelenleg is tanulmányozom a nyelvet. A projekteimen is látható, milyen szinten is állok. Legutóbb a FETCH, és a JSON API-jal ismerkedtem.

    JavaScript

Keretrendszerek

  • Megismerkedtem a Bootstrap 5-ös keretrendszerrel, és nem okoz gondot a használata. Az egyik megbízásomat Bootstrap-pel, és Bootstrap Sablonnal készítem, ahol a változtatásokat is könnyen elvégezem.

    Bootstrap 5

  • Mivel imádok ikonokat használni az oldalaimhoz, a Font Awesome-mal régi rémálmomat szűntette meg. Végre nincs több képként beillesztés, és méretezés, mert ezzel a keretrendszerrel minden ikont könnyen használhatok.

    FontAwsome

Verzió kezelés

  • A GIT és GitHub alap kezelését megtanultamm mint például a STAGING AREA-hoz hozzáadást, a COMMIT-olást, valamint a PUSH-olást és PULL-olást. Tudok klónozni, és forkolni, és egy munkámat megjeleníteni a GitHub Pages-en.
